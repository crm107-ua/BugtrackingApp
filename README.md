# BugtrackingApp

## Project setup
```
npm install
npm install axios
npm install bootstrap
npm install json-server
```

### Compiles and hot-reloads for development
```
json-server --watch data/db.json --port 5000
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
